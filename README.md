# 航天云课堂辅助工具

## 主要功能

1. 专题课程页面生成学习按钮，全自动学习除  ***考试课程***  外的所有课程

2. 课程视频播放页面防止弹窗出现

## 使用方法

0. 油猴安装脚本

1. 专题课程学习：进入专题学习页面，点击“专题内容”上方的开始学习按钮，***全心打工***，等待所有课程播放完毕

2. 单个课程学习：进入课程视频页面，播放视频。***全心打工***，等待视频播放完毕
